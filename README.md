# SWOT Analysis App

An educational web application that teaches users about SWOT (Strengths, Weaknesses, Opportunities, Threats) analysis through an interactive learning experience.

## Features

- **Educational Section**: Comprehensive guide explaining SWOT analysis fundamentals
- **Interactive Quadrants**: Visual representation of SWOT components with hover effects
- **Internal vs External Factors**: Clear distinction between controllable and uncontrollable elements
- **Worked Example**: Real-world scenario demonstrating SWOT application
- **Knowledge Checks**: Simple quizzes to test understanding
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Read through the educational content
4. Take the knowledge check quiz
5. Click "Begin Your SWOT Analysis" (placeholder for future interactive tool)

## Project Structure

- `index.html` - Main educational page
- `styles.css` - Styling for the application
- `script.js` - Interactive functionality
